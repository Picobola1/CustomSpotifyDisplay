# CustomSpotifyDisplay

This is a custom Mp3 player designed to look like a cross between a game boy and a Ipod :) This project uses a Xiao ESP 32 and a 1.8" TFT display. I made this project mostly beacuse i realized that since Parthenon (Athena Award Hackathon) and now i have been 1 project and i desided it about time i make something new!
